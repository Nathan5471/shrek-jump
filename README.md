# Shrek Jump

## Description

Shrek Jump is a platformer game where you use the arrow keys to move around and jump. You goal is to collect all of the coins. Once you collect all of the coins on the screen the level will reset and you speed will increase by 1. You are stuck in this loop untill you reach a speed of 5.
